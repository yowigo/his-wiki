## 基本流程

```
医生开药品医嘱 → 收费 → 药房发药 → 患者取药
```

## 接口组相关

- Plugins.Drug — 药品管理插件
- 上海药事平台对接：`D:\work\his-medical-group\上海药事平台\`

## 相关Schema

- `drug` — 药品数据

## 详细业务流程

### 1. 药品医嘱开具阶段
- 医生开具药品医嘱
- 药品选择（通用名、商品名、规格）
- 用法用量指定
- 用药频次设置
- 疗程天数计算

### 2. 药品审核阶段
- 药师审核医嘱
- 药物相互作用检查
- 过敏史检查
- 剂量合理性检查
- 医保目录匹配

### 3. 收费阶段
- 药品费用计算
- 医保报销比例计算
- 自费部分计算
- 收费确认

### 4. 发药阶段
- 药房接收发药任务
- 药品库存检查
- 药品调配
- 发药确认
- 用药指导

### 5. 取药阶段
- 患者取药
- 药品核对
- 用药说明
- 取药确认
