# 文档说明

## 概要描述

本文档是HIS系统与第三方应用之间数据交互服务开发的标准说明。

## 阅读对象

开发、实施成员、涉及使用基层系统与三方对接的人员。

# 数据交互技术规范

目前发布的Webapi服务为JSON报文格式

## HTTP通讯说明（JSON格式）

通过HTTP+POST方式访问，请求数据通过JSON方式传递

内容类型：application/json; charset=utf-8

请求地址： <http://192.168.21.41:13100/>对应服务地址

{

system\_code:"S0001-自助机，S0002-公众号,S0010-手麻,S0099-其他",

}
