# 文档说明

## 概要描述

本文档是基层系统与第三方应用之间数据交互服务开发的标准说明。

## 阅读对象

开发、实施成员、涉及使用基层系统与三方对接的人员。

# 数据交互技术规范

目前发布的Webapi服务为JSON报文格式

## HTTP通讯说明（JSON格式）

通过HTTP+POST方式访问，请求数据通过JSON方式传递

内容类型：application/json; charset=utf-8

请求地址： <http://192.168.21.41:13100/>对应服务地址

### 请求数据说明:

{

system\_code:"S0001-自助机，S0002-公众号,S0099-其他",

system\_name:"渝北人民医院自助机 ",

各接口参数

}

### 返回数据说明

{

**"code"**:**200**,

**"msg"**:**"查询成功"**,

**"data"**:[]

}

![](data:image/png;base64...)

### 药品库存查询

1、通过机构id、库房id可直接查询药品库存信息。

![](data:image/png;base64...)

2、药品目录查询用于第三方查询药品库存信息使用。

## 药品申领

### 新增药品申领单

1、用于保存药品申领单

![](data:image/png;base64...)

2、注：申领部门、发药部门的数据信息通过“B1002获取部门信息”获取。

1. 新增时若选择领用则申领后走领用流程，需要将申领单转领用单后才可审核和冲销，若不勾选则走移库流程，需要将申领单转移库单才可审核和冲销

### 编辑药品申领单

1、根据药品申领单记录查询申领记录，用于修改药品申领单

![](data:image/png;base64...)

2、注：申领部门、发药部门的数据信息通过“B1002获取部门信息”获取。

3、单据处于未审核状态才可编辑。

### 取消药品申领单

1、用于删除药品申领单

![](data:image/png;base64...)

2、注：单据处于未审核状态时才可删除。
